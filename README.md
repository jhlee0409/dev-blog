# Dev Blog

Next.js와 TypeScript로 구축된 개인 기술 블로그입니다.

## 기술 스택

- **Frontend**: Next.js 15.1.5, React 19.0.0, TypeScript 5.3.3
- **스타일링**: TailwindCSS 4.0.0-alpha.13
- **마크다운**: next-mdx-remote
- **분석**: Vercel Analytics, Speed Insights
- **배포**: Vercel

## 주요 기능

- 반응형 디자인
- 마크다운 기반 블로그 포스팅
- 코드 신택스 하이라이팅
- RSS 피드 지원
- SEO 최적화
- 성능 분석 및 모니터링
